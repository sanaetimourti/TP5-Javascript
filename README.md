# TP JavaScript : Introduction & Évènements

Ce projet a été réalisé dans le cadre du module **Technologies du Web & Web Sémantique** du **Master SDIA** à l'École Normale Supérieure de l'Enseignement Technique (ENSET) de Mohammedia. Il regroupe une série de quatre exercices pratiques visant à maîtriser les bases de JavaScript, la manipulation du DOM et la gestion des évènements.

##  Fonctionnalités et Exercices

Le projet est constitué d'une page HTML unique (`tpjavascript.html`) intégrant les 4 exercices suivants :

* **Exercice 1 : Permutation de variables**
  * Interface contenant deux champs de texte.
  * Bouton permettant d'intervertir le contenu des deux champs en utilisant une variable temporaire en JavaScript.

* **Exercice 2 : Calculatrice Simple**
  * Outil basique permettant d'effectuer les 4 opérations mathématiques élémentaires (+, -, *, /) sur deux nombres saisis par l'utilisateur.
  * Gestion des erreurs (champs vides, division par zéro).

* **Exercice 3 : Calculateur d'IMC (Indice de Masse Corporelle)**
  * Formulaire prenant en entrée le poids (kg) et la taille (m).
  * Calcul automatique de l'IMC et affichage de l'interprétation selon les normes de l'OMS (Corpulence normale, Surpoids, Obésité, etc.).

* **Exercice 4 : Calculatrice Avancée (Scientifique)**
  * Interface complexe reproduite fidèlement selon une maquette en utilisant **CSS Grid**.
  * Intégration de fonctions mathématiques avancées (sin, cos, tan, log, ln, exponentielle, racines carrées).
  * Algorithme basé sur l'évaluation dynamique de chaînes de caractères via JavaScript (`eval()`) avec formatage visuel pour l'utilisateur.

## Technologies Utilisées

* **HTML5** : Structure de la page et des formulaires.
* **CSS3** : Mise en page, design responsive et utilisation avancée de **CSS Grid** pour la calculatrice scientifique.
* **JavaScript (Vanilla JS)** : Logique algorithmique, écoute des évènements (clics) et interaction avec le DOM (`getElementById`, `innerHTML`, `value`).
